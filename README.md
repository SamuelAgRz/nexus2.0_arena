# Nexus 2.0 Notebook Replica

This repo contains a minimal implementation of a Nexus-like multi-agent system in Python.

## Components
- FoundryOpenAIClient (LLM)
- PowerBIClient (DAX execution)
- Agents:
  - Intent Clarifier
  - DAX Developer
  - DAX Validator
  - DAX Executor
  - DAX Result Summarizer
  - Visualization Agent
  - Final Summarizer

## Usage
Open the notebook and configure environment variables.
